# Tic Tac Toe Game (Java)

This is a 3x3 command-line Tic Tac Toe game developed in Java for two players.

## Features
- 3x3 game board
- Player 1 (X) and Player 2 (O)
- Player name input
- Turn based play
- Input validation
- Win and Draw detection
- Replay option

## How to Run

Compile:
javac TicTacToe.java

Run:
java TicTacToe

## How to Play
Enter row and column values between 0 and 2.
Example:
0 1

The game announces winner or draw after each round.
